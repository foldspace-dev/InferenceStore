# Decide first real local adapter: LiteRT-LM

Labels: `area/research`, `area/provider`, `type/decision`, `priority/p0`  
Milestone: `M0 Validation`  
Dependencies: None

## Problem

The MVP must exercise a real local runtime instead of relying only on fake local providers.

## Proposal

Accept LiteRT-LM Android/JVM as the first real local adapter. Defer iOS adapter choice to M3 mobile proof.
